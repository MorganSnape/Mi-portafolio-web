<SubtitleBlog align="left">
  ## ¿Qué es?
</SubtitleBlog>

Diseño Gráfico es un proceso de creación de diseños visuales, que consiste en la composición de imágenes, textos, gráficos, iconos y otros elementos visuales para crear diseños que se adapten a las necesidades y expectativas de los usuarios.

<SubtitleBlog align="left">
  ## ¿Para qué sirve?
</SubtitleBlog>

- Crear diseños visuales atractivos y efectivos
- Diseñar interfaces de usuario
- Diseñar aplicaciones
- Diseñar productos

<SubtitleBlog align="left">
  ## ¿Qué es el diseño visual?
</SubtitleBlog>

El diseño visual es el proceso de creación de diseños visuales, que consiste en la composición de imágenes, textos, gráficos, iconos y otros elementos visuales para crear diseños que se adapten a las necesidades y expectativas de los usuarios.
